# Картотека книг (Telegram Bot)

Fullstack-приложение в виде Telegram-бота для создания и ведения личной картотеки книг с полноценной системой авторизации, админ-панелью и поддержкой донатов через Telegram Stars.

Бэкэнд приложения написан на языке **Python** с использованием библиотеки **pyTelegramBotAPI** и встроенной СУБД **SQLite** для хранения данных пользователей и книг. Вся логика работы с базой данных вынесена в отдельные SQL-запросы внутри кода.

---

## Функционал

* **Классическая авторизация:** Регистрация и вход по уникальному логину (никнейму) и паролю напрямую через интерфейс Telegram. При вводе пароля сообщения автоматически удаляются из чата в целях безопасности.
* **Управление картотекой:** Возможность добавлять книги (название, автор, жанр, описание, статус, оценка от 1 до 5), редактировать описание и удалять свои книги.
* **Гибкий пропуск шагов:** Возможность пропустить ввод описания книги в один клик с помощью Inline-кнопки.
* **Полноценная Админ-Панель:** Просмотр книг всех пользователей системы (с отображением их логинов), возможность удалять чужие книги и принудительно редактировать их описания. Назначение и снятие прав администратора с других пользователей по их логину.
* **Монетизация (Telegram Stars):** Встроенная поддержка нативной оплаты «Звездами» (XTR) для отправки донатов авторам книг.
